## ğŸ“ Key Checkpoints

### Checkpoint 1: Identity Gate
**Location:** After plan selection, before profile building
**Blocks:** Cannot submit to admin without verified identity
**Bypass:** None (mandatory for all)

### Checkpoint 2: Moderation Gate
**Location:** After content save (bio, photos)
**Blocks:** Cannot submit if auto_blocked or auto_flagged
**Bypass:** Fix content until auto_passed

### Checkpoint 3: Completeness Gate
**Location:** Before submit button
**Blocks:** Missing required fields/photos/rates
**Bypass:** Complete all requirements

### Checkpoint 4: Admin Gate
**Location:** Before publication
**Blocks:** Profile stays private until approved
**Bypass:** Admin approval only

### Checkpoint 5: Payment Gate (paid plans only)
**Location:** Throughout journey + post-publication
**Blocks:** Profile goes offline if subscription fails
**Bypass:** Update payment method
